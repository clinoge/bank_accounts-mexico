# Bankaccounts::Mexico

This is a gem providing a YAML formatted database and a simple API over it.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bankaccounts-mexico'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bankaccounts-mexico
